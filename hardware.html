Hardware and headsets

What the work actually runs on. Every category before this one defaults to a handheld phone; this page is about the cases where a headset changes what an installation can ask of a visitor.

Option What it is Cost Learning curve Distribution Offline capable Vibecoding fit Used in
None of the projects behind this atlas run on dedicated hardware yet. Everything so far targets a handheld phone browser, by design: it is the option with zero install friction for gallery visitors and workshop participants. A headset would only make sense for a piece that specifically needs both hands free for an extended, multi-visitor gallery run.
